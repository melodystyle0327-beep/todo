# Node.js 프로젝트

## 시작하기

### 필요한 것
- Node.js가 설치되어 있어야 합니다.

### 설치 및 실행

```bash
# Node.js가 설치되어 있는지 확인
node --version

# 프로젝트 실행
npm start

# 또는
node index.js
```

### 설명
이 프로젝트는 기본 Node.js 프로젝트 구조입니다.
- `index.js`: 메인 진입점 파일
- `package.json`: 프로젝트 설정 및 의존성 관리

서버는 기본적으로 http://localhost:3000 에서 실행됩니다.
